# Detection-of-Exoplanets-using-Machine-Learning

An automated exoplanet detection and classification system that leverages ensemble machine learning algorithms to identify exoplanet candidates from NASA astronomical datasets. The project combines Random Forest, XGBoost, Logistic Regression, and Support Vector Machine (SVM) models to achieve high prediction accuracy while providing user-friendly interfaces through Flask and Streamlit.

---

## Overview

Exoplanets are planets that orbit stars outside our Solar System. Detecting them is a challenging task due to the massive amount of astronomical data generated by missions such as NASA's Kepler and TESS.

This project automates the detection and classification of exoplanets using machine learning techniques. The system processes astronomical datasets, extracts relevant features, trains an ensemble learning model, and automatically classifies planetary candidates as:

- Confirmed Exoplanet
- False Positive

The system runs completely offline and provides real-time monitoring of newly added data.
